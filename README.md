# Veterinary Clinical Data Analysis

## Overview

This project demonstrates foundational data analysis skills using Python and pandas to clean, validate, transform, and analyze veterinary clinical data.

## Skills Demonstrated

- Data Import and Inspection
- Data Cleaning and Transformation
- Missing Value Analysis
- Duplicate Detection
- Filtering and Querying
- GroupBy Aggregations
- Exploratory Data Analysis
- Google Colab Notebook Documentation

## Tools

- Python
- Pandas
- Google Colab

## Key Tasks

- Standardized data values and column naming
- Created calculated fields
- Investigated duplicate records
- Assessed missing data
- Analyzed trends across species, breed, age, and medical history
- Documented findings using markdown and notebook reporting
